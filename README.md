

  Zomato Data Analysis Using Python

This project focuses on performing **Exploratory Data Analysis (EDA)** on Zomato restaurant data to uncover insights about customer preferences, restaurant trends, and rating patterns.

Using Python and data visualization libraries, the analysis explores how factors such as restaurant type, online ordering availability, cost, and customer votes influence ratings and popularity.

---

 Key Highlights

* 📊 Analyzed restaurant categories and their distribution
* ⭐ Explored rating patterns and cleaned inconsistent data
* 🗳️ Identified restaurants with the highest votes
* 📦 Studied cost distribution for two people
* 📱 Compared online vs offline ordering impact on ratings
* 🔥 Visualized insights using graphs and heatmaps

---

## 🛠️ Tools & Technologies

* Python
* Pandas & NumPy
* Matplotlib & Seaborn

---

## 🎯 Objective

The goal of this project is to gain meaningful insights from real-world data and understand customer behavior in the food industry using data analysis techniques.

---

## 📈 Outcome

The project helps in:

* Understanding market trends in the restaurant industry
* Identifying popular restaurant types
* Analyzing factors affecting customer ratings
